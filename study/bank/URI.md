# HTTP

### URI
- URI는 Uniform Resource Identifier로 URL(Resource Locator)과 URN(Resource Name)으로 나눌수 있다.
- URI는 리소스가 있는 위치를 지정하고 URN은 리소스에 이름을 부여한다.
- 사실상 URI는 URL을 포함하기에 비슷한 의미로 쓰인다.
