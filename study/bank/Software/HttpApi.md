## HTTP API

- HTTP API는 다양한 시스템에서 호출해 데이터를 주고 받는다.(주로 JSON 사용) UI가 필요하면 클라이언트가 별도 처리한다.

## SSR(Server side Rendering)
- HTML 최종결과를 서버에서 만들어서 웹 브라우저에 전달
- JSP, 타임리프 -> 백엔드!


  <br>

  
## CSR(Client side Rendering)
- HTML 결과를 자바스크립트를 사용해 웹 브라우저에 동적으로 생성해서 적용
- React, Vue.js -> 프론트엔드!
